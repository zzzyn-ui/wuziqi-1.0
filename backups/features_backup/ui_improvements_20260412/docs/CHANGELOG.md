# 更新日志

本文档记录五子棋在线对战服务器的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.2.0] - 2024-03-01

### 新增
- **残局系统**: 新增残局闯关模式，包含简单/中等/困难三个难度
- **残局记录**: 记录用户残局通关记录和星级评定
- **残局排行榜**: 新增残局通关排行榜
- **好友私聊**: 支持好友之间一对一聊天
- **未读消息**: 新增未读消息数量查询接口

### 改进
- 优化匹配算法，减少匹配等待时间
- 改进棋盘渲染性能
- 增强错误提示信息

### 修复
- 修复观战模式下棋盘状态不同步问题
- 修复断线重连后积分计算错误
- 修复好友列表在线状态显示问题

---

## [1.1.0] - 2024-02-01

### 新增
- **观战功能**: 支持观战正在进行的对局
- **对局回放**: 支持历史对局完整回放
- **游戏邀请**: 支持向好友发送游戏邀请
- **房间系统**: 支持创建和加入私人房间
- **个人收藏**: 支持收藏精彩对局

### 改进
- 优化 WebSocket 连接稳定性
- 改进 ELO 积分算法
- 增加匹配超时机制

### 修复
- 修复匹配队列偶尔卡死的问题
- 修复聊天消息丢失问题
- 修复排行榜数据刷新延迟

---

## [1.0.1] - 2024-01-15

### 修复
- 修复 JWT token 过期处理问题
- 修复数据库连接池泄露问题
- 修复密码重置功能异常
- 修复移动端页面布局问题

### 改进
- 增加服务器监控指标
- 优化日志输出格式
- 改进错误处理机制

---

## [1.0.0] - 2024-01-01

### 新增
- **用户系统**: 用户注册、登录、认证
- **匹配系统**: 基于 ELO 积分的智能匹配
- **对战功能**: 双人实时在线对战
- **人机对战**: 支持 PvE 模式
- **聊天系统**: 公共聊天室
- **好友系统**: 添加好友、好友列表
- **排行榜**: 积分排行榜
- **对局记录**: 历史对局记录查询
- **积分系统**: ELO 积分计算
- **等级系统**: 基于积分的等级划分

### 技术栈
- 后端: Java 17 + Netty 4.1
- 数据库: MySQL 8.0
- 缓存: Redis 7.0
- 认证: JWT
- 协议: WebSocket + HTTP REST API

## 版本说明

### 版本号格式
- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向下兼容的功能新增
- **修订号**: 向下兼容的问题修复

### 变更类型
- **新增**: 新功能
- **改进**: 现有功能的改进
- **修复**: 问题修复
- **移除**: 功能移除
- **安全**: 安全相关修复
- **弃用**: 即将移除的功能
