📜 五子棋游戏规则

了解规则，享受对弈乐趣

基本规则

  • 棋盘：15×15 的方形棋盘，共 225 个交叉点
  • 棋子：黑白双方各持一组棋子
  • 先行：黑棋先行，双方轮流落子
  • 落子：棋子落在交叉点上，落子后不可移动
  • 获胜：先在横、竖、斜任意方向连成五子者获胜

胜利条件

五连珠 - 在任意方向（横、竖、斜）连续形成五颗同色棋子即获胜

  • 横向五连：○○○○○
  • 纵向五连：垂直方向连续五颗
  • 斜向五连：左斜或右斜连续五颗

游戏模式说明

人机对战：与AI对弈，可选择简单/中等/困难三种难度

在线对战：与真实玩家对弈，分为休闲模式（无积分）和竞技模式（有积分）

本地对战：双人面对面游戏，适合朋友聚会

经典技巧

五子棋有几种常见的制胜技巧：

  • 活三：两端都空的三连，下一步可成活四
  • 冲四：只有一端空的四连，威胁对方必须防守
  • 四三杀法：同时形成冲四和活三，对方无法同时防守
  • VCF（连续冲四）：连续冲四直至获胜

中国五子棋段级位制（2025年版）

根据国家体育总局棋牌运动管理中心规定：

段位等级（9个等级）

从高到低：九段、八段、七段、六段、五段、四段、三段、二段、初段

级位等级（10个等级）

从高到低：一级、二级、三级、四级、五级、六级、七级、八级、九级、十级

获得途径：

  • 通过全国五子棋邀请赛等国内大赛获得相应名次
  • 段位称号由中国棋院统一颁发
  • 2025年版取消了最高授予六段的限制，全面开放至九段

平台积分与段位

超时判罚规则

思考时间限制

为保证对局流畅，每步棋有2分钟的思考时间

  • 单步超时：一方超过2分钟未落子，判该方负
  • 总时间限制：每方总思考时间为5分钟
  • 时间显示：对局界面实时显示双方剩余时间
提示：请在倒计时结束前完成落子，超时将自动判负！
